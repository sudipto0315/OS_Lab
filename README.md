
# OS Lab



## Usage

To navigate through this project go through this path

```path
  ~OS_LAB/<Assignment_Number>/<question_no.c>
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/sudipto0315/OS_Lab.git
```

Go to the project directory

```bash
  cd OS_Lab
```
